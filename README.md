* 📦 Instalação

** Clone o repositório:

git clone https://github.com/paulothiagosantos-code/pedra_papel_tesoura.git
cd pedra_papel_tesoura

* Instale as dependências:

pip install -r requirements.txt

* ▶️ Execução

** Execute o programa principal com o comando:

python main.py
